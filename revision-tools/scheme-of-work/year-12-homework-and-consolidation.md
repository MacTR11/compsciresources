# Year 12 (L6) — Homework & Consolidation Plan, 2026/27

Homework and consolidation tasks to **round out each week's new content** and **build confidence in previously‑covered topics** through deliberate spaced retrieval and interleaving. Aligned to the L6 CMP Scheme of Work and designed for the **~4.5 hrs/week** of independent study (IST/VESPA).

## How this is designed
- **Homework** = consolidates the **week's new content** (a printable quiz + a focused skills task + the programming strand).
- **Consolidation** = *rounds out the week* **and revisits earlier topics** on a spacing schedule (1 day → 1 week → 1 month → cumulative). This is the part that **builds confidence in prior topics** and protects against forgetting.
- Everything links to resources already in this repo, so tasks are zero‑prep to set.

## Notes for the teacher
- **Differentiation:** quizzes have Section A (recall) → C (exam‑style); set Section A for all, B–C for stretch. Workbook tasks are tagged ★/★★/★★★.
- **Low‑prep marking:** every quiz, worksheet and mini‑paper has a **full answer key / mark scheme**, so they're self‑markable or cover‑teacher‑friendly.
- **The confidence loop:** the *Consolidation* column is the spaced‑retrieval spine — it revisits each topic at ~1 week, ~1 month and cumulatively, which is what moves earlier content into long‑term memory and steadies confidence before each Key Assessment/CAP.
- **Adjust freely:** swap a mini‑paper for flashcards if time is short; the principle (new content in Homework, prior content in Consolidation) is what matters.
