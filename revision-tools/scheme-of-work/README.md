# Scheme of Work — Homework & Consolidation

Ready-to-use **homework** and **consolidation** tasks for every week of the **Year 12 (L6) 2026/27** OCR Computer Science scheme of work, built to round out each week's new content **and** spiral back through earlier topics for confidence.

## The design in one line
**Homework = the week's new content** (a printable subtopic quiz + a focused skills task + the programming strand). **Consolidation = round-out + spaced retrieval of earlier topics** (the spine that builds lasting confidence before each Key Assessment/CAP).

Every task points to a resource already in this repo (subtopic quizzes, worksheets, flashcards, mini-papers, knowledge organisers, programming workbook, mock papers, NEA guide), so it's zero-prep to set and self-markable.

> To paste into a Google Sheet/Doc SOW: open the CSV, copy the **Homework** and **Consolidation** columns into the matching columns of your SOW. To regenerate the PDF after edits: `python3 revision-tools/build-pdfs.py revision-tools/scheme-of-work revision-tools/printable-pdfs/scheme-of-work`.
