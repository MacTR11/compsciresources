# Medium-Term Plan & Lesson Resource Map — Year 12 (L6), 2026/27

Maps each week of the **L6 CMP scheme of work** to the resources in this toolkit: which **lesson PowerPoint** to teach from, the **in-class / practice** resources, and the **assessment** point. Pair this with the *Homework & Consolidation* plan (separate file) — this map is the **in-lesson** side.

**Resource key:** *Deck x.y.z* = `Teacher-Toolkit/Lesson-PowerPoints/x.y.z…pptx` · *Quiz/Worksheet/KO/Mini-paper* = the matching files in `Word-Documents/` or `Printable-PDFs/` · *Markbook* = `Teacher-Toolkit/Markbook/H446-Markbook.xlsx`.

> Each lesson deck already contains: objectives · a Do-Now retrieval starter (with answers) · key terminology · content slides · a worked example · a task + stretch challenge · an exit ticket (with answers). Adapt freely.

### Using the markbook
Record each **KA1–8** and **CAP1–5** result in `H446-Markbook.xlsx`: enter the max mark in the yellow row, type raw marks, and the sheet auto-calculates each student's percentage, **grade** (against editable boundaries) and the **class average** per assessment — with RAG colour. Use it at each feedback lesson to target intervention.
