**3. [3]** — Trace of `power(2, 4)` (frames pushed, then popped LIFO):
```
power(2,4) = 2 * power(2,3)
power(2,3) = 2 * power(2,2)
power(2,2) = 2 * power(2,1)
power(2,1) = 2 * power(2,0)
power(2,0) = 1            (base case, exp = 0)
```
Unwinding: 2*1 = 2 → 2*2 = 4 → 2*4 = 8 → 2*8 = **16**.
- 1 mark: base case returns 1 at `exp = 0`.
- 1 mark: correct returns shown at each level (2, 4, 8, 16) / correct multiplication on unwind.
- 1 mark: final answer **16**.
