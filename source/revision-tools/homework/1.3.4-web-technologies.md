**3. [2]** 1 mark: a web crawler/spider automatically visits/reads web pages and follows the links it finds to discover further pages. 1 mark: the content/keywords found are passed to the search engine which builds an index mapping keywords to the pages they appear on (for fast retrieval/ranking).

**4. (a) [2]** 1 mark: each incoming link counts as a "vote" for the page's importance. 1 mark: a vote/link from a high-ranking page carries more weight / passes on more rank than one from a low-ranking page, so it raises the target page's rank more.

**(b) [2]** 1 mark: iterative means each page's rank is recalculated repeatedly using the latest rank values of the pages that link to it. 1 mark: converges means the process stops when the rank values stop changing significantly between iterations (settle to stable values).

**5. (a) [1]** Any one: instant feedback for the user without waiting for a server round trip / reduces load on the server because no request is needed for a simple check.
