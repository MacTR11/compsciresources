1 mark: `SELECT FirstName, Surname FROM Member`; 1 mark: `WHERE MembershipType = 'Junior'`; 1 mark: `ORDER BY Surname DESC` (must include DESC).

**3. [3]** Sample answer:
```sql
SELECT Member.Surname, Booking.CourtNumber, Booking.BookingDate
FROM Member
JOIN Booking ON Member.MemberID = Booking.MemberID;
```
1 mark: SELECT lists Surname, CourtNumber and BookingDate; 1 mark: FROM/JOIN names both tables; 1 mark: correct `ON Member.MemberID = Booking.MemberID`. (Accept table aliases or an implicit join in the WHERE clause.)
